# AI Powered Resume Predictor& Skill Gap Detector

## Overview
An AI-powered web application that analyzes resumes, extracts skills, calculates ATS scores, and identifies skill gaps.

## Features
- Resume Upload
- Skill Extraction
- ATS Score
- Skill Gap Analysis
- Recommendations

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- SpaCy
- Matplotlib
- Seaborn

## Project Goal
To help students improve their resumes and become placement-ready.
